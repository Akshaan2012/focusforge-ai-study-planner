# FocusForge AI Study Planner

FocusForge is a quick, polished static web app that turns exam topics into a day-by-day study sprint.

## Features

- Subject, topic, deadline, daily-hour, and confidence inputs
- Auto-generated daily study timeline
- Priority topic stack
- Practice quiz prompts
- Copyable plan summary
- Responsive UI for desktop and mobile

## Run

Open `index.html` in a browser.

No build step or package install is required.
